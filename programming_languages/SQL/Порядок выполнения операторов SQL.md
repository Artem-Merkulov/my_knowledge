# Порядок выполнения операторов в SQL

|		Рандом				|Синтаксический порядок	|Порядок выполнения запроса|
|:------------- |:---------------:| -------------:|
FROM					|SELECT					|FROM
HAVING					|FROM					|WHERE
SELECT					|WHERE					|GROUP BY
ORDER BY				|GROUP BY				|HAVING
GROUP BY				|HAVING					|SELECT
WHERE					|ORDER					|ORDER
